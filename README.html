Daisy Content Scroller

A lightweight horizontal content scroller that can be easily added to your web site or application.

Features

  • Extensible HTML/CSS interface.
  • Compatible with Firefox 3.6+, Chrome, Safari 5+, Opera, and Internet Explorer 7+ web browsers.
  • Easy to set-up and customize.
  • Supports multiple scoller instances on a single document.
  • Supports HTML, Flash objects and image content.
  • Fast and lightweight (jQuery plug-in *only 3.2 kB)

Check out the demo provided with this package.

Installation

  1. Download the latest sources to your computer using a web browser.
  2. Extract the contents of the .zip into a folder on your local computer.
  3. Upload the folder with the following files to your web site.
Filename Role
jquery.daisy.scroller.min.js The main script to be included from within your HTML document.
jquery.daisy.scroller.min.css This style sheet that defines the "look & feel" of the content scroller.

Set-up

Easy as defining the target element #container using the jQuery selector. An optional callback can also be defined for post-processing of nodes on scrolling events.

$(selector).DaisyScroller(options, callback);

Source Code

Add the following Javascript between the <head></head> tags of your HTML document.

<script src="http://www.google.com/jsapi"></script>
<script>
	google.load('jquery','1.7.1');
	google.load('jqueryui','1.8.16');
</script>

<link rel="stylesheet" type="text/css" href="/path/to/jquery.daisy.scroller.min.css">

<script src="/path/to/jquery.daisy.scroller.min.js"></script>
<script>
$(document).ready(function() {
	$('#container').DaisyScroller({
		scrollSpeed : 1000
	});
});
</script>

Add the following HTML code to your document. Please refer to the demo.css for element overrides. 

<div id="container">
	<button class="daisy_button_last" type="button" title="Last">&lt;&lt; Last</button>

	<div class="daisy">
		<div>Custom HTML</div>
		<div>Custom HTML</div>
		<div>Custom HTML</div>
		..
	</div>

	<button class="daisy_button_next" type="button" title="Next">Next &gt;&gt;</button>
</div>

Plug-in Settings

The following options can be passed to the plug-in main function as JSON

Option Description Default Value
scrollEasing animation to be used when scrolling content easeOutCirc
scrollSpeed speed of the scroller in milliseconds 400
showCounter show the element counter (1 of <total elements>) false

Click here for more information about the available jQuery UI Easing effects.

Public Methods

The following methods can be passed to the plugin-in main function after successful initialization.

Method Description
reset reset the scroller back to its original start position

Callback Processing

When a callback function is defined a response object of scroller nodes is returned. The following function corresponds to the example provided above.

function doSomething(obj) {

	// post-process event
	alert(obj.length);
}
